# KC-GCODE-GENERATOR
generate gcode files for machine at king coolers 



How to: 

1. Open file in excel
2. enable macros
3. open macro option (this may or may not show the module files if not just import them from the modules folder)
4. ***IMPORTANT*** MAKE SURE YOU ARE IN SHEET 1 BEFORE RUNNING ANY MACRO NEVER RUN OUTSIDE OF SHEET 1
5. run macros 2, 4, 5, or 6
6. this will generate a folder on your desktop
7. YOU MAY HAVE ISSUES GENERATING IF YOUR "mainFolderPath" isnt set right... mine is set to \onedrive\desktop ||| yours may be just \desktop. the environ is already set to the user profile that you are logged into.
